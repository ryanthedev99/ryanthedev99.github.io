Ryan's Battery API

Let's learn how to use this addon!

First, let's add this in your <head></head> tag

<script src="battery-level.js"></script>

Now in your body, with your main elements, add a <script></script> tag

Inside here, we will add a battery listener

const myListener = new batteryListener();

You can now access all variables of the live listener, like level, charging, and the float form of the level. BUT, make sure to add your ready event to make sure your variables are ready to access.


let currentCharge = 0;
let currentChargeFloat = 0.00;
let isCharging = false;
myListener.ready.then(() => {
    currentCharge = myListener.level;
    currentChargeFloat = myListener.levelfloat;
    isCharging = myListener.charging;
})

Want to preview your listener variables?


// myListener.ready.then(() =>) { ...
// set variables ...

// log the variables
console.log(currentCharge)
console.log(currentChargeFloat)
console.log(isCharging)

// example output:
// 58
// 0.58
// true

How to add event listeners for live updates

const myListener = new batteryListener() // create a listener

// these events are for updates and DO NOT need ready events.
myListener.addEventListener("levelupdate", funtion() {
    // event listeners for level update (58 - 59) OR (58 - 57)
    console.log(`BATTERY STATUS: %${myListener.level}`)
})

myListener.addEventListener("chargeupdate", funtion() {
    // event listeners for charge updates (true - false) OR (false - true)
    console.log(`CHARGING STATUS: ${myListener.charging}`)
})
